# React Combobox

React combobox is simple implementation of a combobox component.

Stack:

- ReactJS
- Typescript
- Vite

## Run locally

```
pnpm install
pnpm run dev
```

[http://localhost:3000](http://localhost:3000)

## Demo version

Demo version is deployed on Vercel [combobox-react](https://react-combobox-bice.vercel.app/)
